# ☕ Language

Aquí eliges el lenguaje de programación que utilizará tu proyecto Spring Boot.

---

# 📌 Opción recomendada

Normalmente se utiliza:

```text
Java
```

---

# 🧠 ¿Por qué Java?

Spring Boot fue creado principalmente para trabajar con Java.

Por eso:

- tiene mejor compatibilidad,
- más documentación,
- más ejemplos,
- y una comunidad enorme.

---

# 🚀 Ventajas de usar Java con Spring Boot

- ☕ Lenguaje sólido y muy popular.
- 🌍 Altamente utilizado en empresas.
- 🧩 Excelente integración con Spring.
- 📚 Muchísimos recursos para aprender.
- 🔒 Muy estable para aplicaciones Backend.

---

# 💡 Consejo para principiantes

Si estás empezando con Spring Boot:

✅ Aprende primero usando Java.

Después podrás explorar otras opciones compatibles como Kotlin.
