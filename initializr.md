# Spring Initializr
## 🚀 El generador de proyectos de Spring Boot

Si los Starters son los kits de piezas,
Spring Initializr es la fábrica que te entrega la caja completa,
lista para abrir y comenzar a trabajar.

Es una herramienta web que genera automáticamente la estructura básica de un proyecto Spring Boot.
