# ⚡ ¿Qué hace Spring Initializr?

Con un solo clic:

- 📁 genera la estructura del proyecto,
- 🌱 configura Spring Boot,
- 📚 agrega dependencias,
- y 📦 descarga un proyecto completamente funcional.

# 🚀 Resultado

Obtienes una aplicación lista para:

- compilar,
- ejecutar,
- y comenzar a programar inmediatamente.
