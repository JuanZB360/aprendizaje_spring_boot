# ❓ ¿Para qué sirve?

Antes de Spring Initializr,
crear un proyecto Java desde cero era un proceso largo y tedioso.

Había que:

- 📁 crear carpetas manualmente,
- 📚 buscar dependencias compatibles,
- ⚙️ configurar archivos complejos,
- 📥 instalar librerías una por una,
- 🔄 evitar conflictos entre versiones.

---

# 💡 Idea clave

El objetivo principal es:

> ahorrar tiempo y evitar errores de configuración.
