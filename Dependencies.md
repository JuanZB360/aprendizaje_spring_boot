# 📚 La sección de Dependencies
## Los Starters

Aquí seleccionas las funcionalidades que tendrá tu aplicación Spring Boot.

En lugar de buscar librerías manualmente,
simplemente eliges los Starters que necesitas.

Spring Boot se encargará automáticamente de:

- descargar dependencias,
- configurar librerías,
- y preparar el proyecto para funcionar.

---

# ⚙️ ¿Qué hacen las Dependencies?

Las Dependencies agregan funcionalidades listas para usar dentro de la aplicación.

Cada Starter incluye:

- librerías principales,
- configuraciones automáticas,
- y componentes compatibles entre sí.

---

# 🚀 Ventajas de usar Starters

## ⚡ Desarrollo más rápido

No necesitas instalar librerías una por una.

---

## 🧩 Compatibilidad automática

Spring Boot se asegura de que todas las versiones funcionen correctamente juntas.

---

## 🛠️ Menos configuración manual

La mayoría de configuraciones se realizan automáticamente.

---

# 📌 Ejemplos comunes

| Necesidad | Dependency |
|---|---|
| 🌐 Crear APIs | `Spring Web` |
| 🗄️ Base de datos | `Spring Data JPA` |
| 🔒 Seguridad | `Spring Security` |
| ✅ Validaciones | `Validation` |
| 🎨 Plantillas HTML | `Thymeleaf` |

---

# 🧠 Idea rápida para recordar

Los Starters son como:

## 📦 “Packs de funcionalidades”

Tú eliges lo que necesitas
y Spring Boot instala todo automáticamente.

---

# 💡 Ejemplo sencillo

Si agregas:

```text
Spring Web
```

Spring Boot prepara automáticamente:

- rutas HTTP,
- APIs REST,
- servidor embebido,
- manejo de JSON,
- y controladores.

Todo listo para usar.

---

# 📌 Resumen rápido

| Concepto | Significado |
|---|---|
| Dependencies | Funcionalidades agregadas al proyecto |
| Starter | Paquete preconfigurado de librerías |
| Beneficio principal | Ahorrar tiempo y reducir configuración |
| Ejemplo común | `Spring Web` |
