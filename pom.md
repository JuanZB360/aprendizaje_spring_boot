# 📄 El Manifiesto
## `pom.xml`

Es el archivo principal que utiliza Maven para administrar el proyecto.

Aquí se definen:

- dependencias.
- Starters.
- plugins.
- versiones.
- y configuraciones generales.

---

# 🧩 ¿Qué contiene?

`pom.xml` utiliza etiquetas XML para listar las librerías necesarias del proyecto.

Por ejemplo:

- Spring Web.
- Spring Data JPA.
- Spring Security.
- entre otras dependencias.

---

# 🎯 Propósito principal

Define qué librerías traerá el “kit” de tu aplicación.

Maven utiliza este archivo para:

- 📥 descargar dependencias.
- 🔄 controlar versiones.
- ⚙️ construir el proyecto.
- 🛠️ evitar conflictos entre librerías.

---

# 🚀 ¿Por qué es importante?

Gracias al `pom.xml`:

- no necesitas instalar librerías manualmente.
- Spring Boot mantiene compatibilidad entre versiones.
- la configuración del proyecto se vuelve mucho más simple.

---

# 💡 Idea rápida

funciona como el:

#### 📋 Inventario oficial del proyecto
