# 📦 El Almacén de Configuración
## `src/main/resources`

Aquí se almacenan los archivos de configuración y recursos del proyecto.

Es el lugar donde defines cómo se comportará la aplicación Spring Boot.

---

# ⚙️ `application.properties`

Es el archivo de configuración más importante del proyecto.

Aquí defines aspectos como:

- 🌐 el puerto del servidor.
- 🗄️ la conexión a la base de datos.
- 🔒 configuraciones de seguridad.
- 📚 propiedades de Spring.
- y variables generales de la aplicación.

---

# 🧩 ¿Por qué es importante?

Este archivo permite configurar el comportamiento de componentes como:

- `@Entity`
- `@Repository`
- `@Service`
- configuraciones internas de Spring Boot.

---

# 💡 Ejemplos comunes

Aquí normalmente se configura:

- el usuario y contraseña de la base de datos.
- la URL de conexión.
- el puerto del servidor.
- opciones de ejecución.

---
