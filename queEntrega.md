# 📦 ¿Qué entrega?

Spring Initializr descarga un archivo `.zip`
que contiene todo lo necesario para iniciar el proyecto.

---

# 📁 Incluye

- estructura de carpetas,
- archivos de configuración,
- dependencias,
- servidor integrado,
- configuración inicial del proyecto.

---

# 🚀 Resultado

Obtienes un proyecto listo para:

- abrir en tu IDE,
- ejecutar,
- comenzar a programar inmediatamente.
