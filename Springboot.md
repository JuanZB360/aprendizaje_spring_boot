# 🚀 ¿Qué es Spring Boot?

Para entender Spring Boot,
primero debemos saber que existe algo llamado:

# 🌱 Spring Framework

Spring Framework es un ecosistema muy grande y potente para Java.

El problema es que antes era complicado de configurar.

Había que:

- escribir muchas configuraciones,
- manejar dependencias manualmente,
- y preparar varios archivos técnicos antes de comenzar.

---

# ⚡ ¿Qué es Spring Boot?

Spring Boot nació para solucionar ese problema.

Es una herramienta construida encima de Spring Framework que permite crear aplicaciones listas para funcionar con mucho menos esfuerzo.

---

# 🧱 Los tres pilares de Spring Boot

---

## ⚙️ 1. Configuración Automática
### *(Auto-configuration)*

Spring Boot analiza tu proyecto automáticamente.

Por ejemplo:

Si detecta una dependencia de base de datos,
entiende que probablemente quieres conectarte a una base de datos y configura lo básico por ti.

---

## 🧠 2. Sistema “Opinado”
### *(Opinionated)*

Spring Boot toma decisiones inteligentes automáticamente.

Por ejemplo:

- selecciona versiones compatibles,
- aplica buenas prácticas,
- y evita configuraciones innecesarias.

Todo pensado para que el proyecto funcione desde el inicio.

---

## 🌐 3. Servidor Embebido

Antes era necesario instalar servidores externos como Tomcat.

Spring Boot ya lo incluye dentro de la aplicación.

Por eso el proyecto puede ejecutarse directamente como un único archivo `.jar`.

---

# 🚀 ¿Por qué usamos Spring Boot?

---

## ⚡ Acelera el desarrollo

Permite crear aplicaciones funcionales en minutos.

---

## 🧩 Reduce la complejidad

Los Starters y las anotaciones hacen gran parte del trabajo pesado.

---

## 🔄 Gestiona dependencias automáticamente

Spring organiza componentes y configuraciones por ti.

---

# 🏗️ Analogía para entenderlo

| Tecnología | Analogía |
|---|---|
| Java | Los ladrillos |
| Spring Framework | Un depósito gigante de herramientas |
| Spring Boot | Una casa prefabricada lista para usar |

Spring Boot ya trae:

- tuberías,
- electricidad,
- paredes,
- y estructura preparada.

Tú solo te concentras en construir la lógica de tu aplicación.

---

# 💡 Resumen para el Junior

Spring Boot es una herramienta que permite crear aplicaciones Java de forma rápida y profesional usando:

- 📦 Starters,
- 🏷️ anotaciones,
- ⚙️ configuración automática,
- y 🚀 servidores integrados.

Así el desarrollador puede enfocarse más en programar y menos en configurar.
