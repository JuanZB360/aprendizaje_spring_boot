# ⚙️ Los Ejecutables
## `mvnw` y `mvnw.cmd`

Estos archivos son conocidos como:

# 📦 Maven Wrapper

Su objetivo es facilitar la ejecución del proyecto sin necesidad de instalar Maven manualmente.

---

# 🚀 ¿Para qué sirven?

Ayudan a:

- ⚡ acelerar el desarrollo,
- 🔄 garantizar la misma versión de Maven,
- 🛠️ evitar problemas de configuración entre equipos.

---

# ⚙️ ¿Qué hacen exactamente?

Actúan como un “puente” entre el proyecto y Maven.

Si Maven no está instalado en la computadora:

- lo descargan automáticamente.
- configuran la versión correcta.
- ejecutan las tareas necesarias.

---

# 🧱 Tareas que pueden ejecutar

- 🧹 limpiar el proyecto,
- ☕ compilar código,
- 🧪 ejecutar pruebas,
- 📦 generar el archivo `.jar`.

---

# 💻 Diferencia entre ambos archivos

| Archivo | Sistema operativo |
|---|---|
| `mvnw` | Linux / Mac |
| `mvnw.cmd` | Windows |

---

# 💡 Idea rápida

Gracias al Maven Wrapper:

✅ todos los desarrolladores usan exactamente la misma versión de Maven.

Eso evita errores y diferencias entre computadoras.
